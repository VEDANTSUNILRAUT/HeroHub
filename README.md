# HeroHub

HeroHub is a modern Android application built using Jetpack Compose and Clean Architecture to explore superhero data from a public API.

---

## Tech Stack

- Kotlin (JDK 17)
- Jetpack Compose (Material 3)
- MVVM Architecture
- Clean Architecture
- Hilt (Dependency Injection)
- Retrofit (planned)

---

## Project Status

Phase 1 completed — Project Setup & Base Architecture

## Phase 1: Project Setup

The initial foundation of the application has been established with a focus on scalability, maintainability, and modern Android development practices.

### Included

- Project configuration with Kotlin and JDK 17
- Jetpack Compose setup with Material 3

- Base Gradle configuration aligned with modern standards
