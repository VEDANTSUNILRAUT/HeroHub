# HeroHub 🦸

HeroHub is a modern Android application built using Jetpack Compose and Clean Architecture to explore superhero data from a public API.

---

## 🚀 Tech Stack

* Kotlin
* Jetpack Compose
* MVVM
* Clean Architecture
* Retrofit (planned)
* Hilt (planned)

---

## 🏗️ Project Status

🚧 Initial setup in progress

## 📌 Goal

To build a production-ready Android app demonstrating scalable architecture and modern development practices.
